Food spectrographs are used in chemometrics to classify food types,
a task that has obvious applications in food safety and quality
assurance. Each class of this data set is an extra virgin olive oil
from alternative countries.

Further information can be found in the <a
href="http://www.ncbi.nlm.nih.gov/pubmed/14518931">original
paper</a>. The data was first used in the time series
classification literature in <a
href="https://archive.uea.ac.uk/~ajb/Papers/BagnallSDM2012.pdf">this
paper.</a>"
